# LLM4AscendC 生成 / Agent / 知识库 可选依赖（requirements-generation.txt）
#
# 两种用法：
#  A) 仅生成：勿与评测混用不同 torch 时，按下面「步骤 2」先装 CPU torch，再装本文件。
#  B) 与评测同一 conda（如 czh_environ）：先 pip install -r requirements.txt，再装 CANN 配套 torch_npu
#     wheel，最后执行：
#       pip install -r requirements-generation.txt -c constraints-czh-unified.txt
#     以保持 torch==2.1.0、numpy==1.26.0 不被 sentence-transformers 等升级（见 README §1.5.3）。
# =============================================================================
#
# --- 建议安装顺序（仅生成、或尚未合并评测栈时；昇腾服务器尽量避免拉 NVIDIA CUDA 大包）---
#
# 1) 激活环境：
#      # 若未执行过 conda init，bash 里需先加载 conda（路径按你本机 Miniconda/Anaconda 安装位置改）：
#      source /root/miniconda3/etc/profile.d/conda.sh
#      conda activate czh_environ
#      # 若不想 source，也可全程用绝对路径，例如：
#      #   /root/miniconda3/envs/czh_environ/bin/pip install ...
#      #   /root/miniconda3/envs/czh_environ/bin/python ...
#
# 2) 先单独安装 CPU 版 PyTorch，再装本文件其余依赖（让 pip 不再去装 nvidia-* CUDA wheel）：
#      pip install -U pip setuptools wheel
#      # 优先尝试官方 CPU 索引（若下载慢或卡住，可换官方 PyPI 或临时关掉镜像）：
#      pip install torch --index-url https://download.pytorch.org/whl/cpu
#      # 若上述在 aarch64 上不可用或仍拉 CUDA，可改为（以实际能装上的 CPU 包为准）：
#      #   pip install torch
#      # 装完后可用：python -c "import torch; print(torch.__version__, torch.cuda.is_available())"
#      #   期望 cuda_is_available 为 False，且 pip list 中无 nvidia-cudnn-cu* 等。
#
# 3) 在仓库根目录 LLM4AscendC/ 下安装本清单：
#      pip install -r requirements-generation.txt
#
# 4) BGE-M3 嵌入模型（需自行下载；默认路径 generation/agent/models/bge-m3，可用 BGE_M3_PATH 覆盖）：
#      pip install modelscope
#      mkdir -p generation/agent/models/bge-m3
#      # 推荐（Python API，各版本一致）：
#      python -c "from modelscope import snapshot_download; snapshot_download('BAAI/bge-m3', local_dir='generation/agent/models/bge-m3')"
#      # 若使用 CLI，请先 modelscope download -h 查看当前版本的参数名（常见为 model_id + --local_dir）
#      魔搭模型页: https://www.modelscope.cn/models/BAAI/bge-m3
#
# 5) 建 PDF 向量库（可选，跑 generation/agent/db_related/build_knowledge_base*.py 时需要）：
#      本文件已包含 unstructured / pdfminer.six / pypdf；建库前确保 Knowledge/ 下已有 PDF。
#
