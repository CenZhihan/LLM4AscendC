公式：output = x1 + value * (x2 / x3)

输入 Shape：
  x1: [1024]
  x2: [1, 1, 2, 1]
  x3: [1]

判断：
  ✅ 逐元素运算（+, *, /）
  ✅ Shape 不同
  ✅ 可兼容

推导：
  Step 1 - 维度对齐（补1到4维）：
    x1: [1024]      → (1, 1, 1, 1024)
    x2: [1,1,2,1]   → (1, 1, 2, 1)
    x3: [1]         → (1, 1, 1, 1)

  Step 2 - 逐维取 max：
    dim0: max(1, 1, 1) = 1
    dim1: max(1, 1, 1) = 1
    dim2: max(1, 2, 1) = 2
    dim3: max(1024, 1, 1) = 1024

  Step 3 - 输出：
    [1, 1, 2, 1024]
