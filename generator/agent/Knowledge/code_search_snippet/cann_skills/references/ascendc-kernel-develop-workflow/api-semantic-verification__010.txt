开始：需要使用某个 API
    ↓
【步骤 1】查阅官方文档
    ├─ 确认 API 签名
    ├─ 确认适用场景
    └─ 确认限制条件
    ↓
【步骤 2】确认数据布局
    ├─ 内存排列方式
    ├─ 对齐状态
    └─ 维度信息
    ↓
【步骤 3】验证匹配性
    ├─ 数据布局匹配？
    │   ├─ 否 → ❌ 换 API 或调整数据
    │   └─ 是 ↓
    ├─ 限制条件满足？
    │   ├─ 否 → ❌ 换 API 或调整数据
    │   └─ 是 ↓
    └─ 有更好选择？
        ├─ 是 → 考虑换 API
        └─ 否 ↓
【步骤 4】确认选择
    ↓
【步骤 5】记录验证结果
    ↓
✅ 使用 API
