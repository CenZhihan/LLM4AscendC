主 Agent
   │
   ├─ 1. 决定分支优先级
   │   └─ 根据设计文档确定分支顺序（简单 → 复杂）
   │
   ├─ 2. Task 工具调度 通用 Agent（实现者+自审）
   │    └─ 步骤 0-5: 读取环境、设计文档、实现代码
   │    └─ 步骤 6: 审视代码 ⚠️ 必须执行
   │    └─ 步骤 7: 编译验证
   │    └─ 步骤 8: 测试验证（3 个基础用例）
   │    └─ 步骤 9: 返回完整报告
   │
   ├─ 3. Task 工具调度 通用 Agent（验收者）← 独立调用
   │    └─ 代码审查
   │    └─ 复跑通用 Agent(实现者) 的测试用例
   │    └─ 返回验收报告（评分）
   │
   ├─ 4. 检查验收结果
   │   ├─ 评分 >= 8.5 + 测试通过 → ✅ 通过
   │   └─ 评分 < 8.5 或测试失败 → 启动【新通用 Agent】修复
   │                           └─ 读取设计文档 + 理解问题
   │                           └─ 针对问题修复代码
   │                           └─ 编译 + 测试验证
   │                           └─ 循环步骤 3-4（最多 3 次）
   │
   └─ 5. 重复步骤 2-4 直到所有分支完成
