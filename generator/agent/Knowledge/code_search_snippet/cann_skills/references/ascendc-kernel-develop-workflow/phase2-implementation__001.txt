   │
   ├─ Task 工具调度 → 通用 Agent（实现者+审视代码）
   │                    └─ 步骤 0-8: 实现+审视代码+编译+测试 → 返回报告
   │
   ├─ Task 工具调度 → 通用 Agent（验收者）← 独立调用
   │                    └─ 审查代码 + 复跑测试 → 返回验收报告（评分）
   │
   └─ 根据评分决定：
        ├─ 评分 >= 8.5 + 测试通过：✅ 通过
        └─ 评分 < 8.5 或测试失败：启动【新通用 Agent】修复
                                 └─ 遵循设计文档，针对问题修复
                                 └─ 步骤 0-8 + 编译测试
