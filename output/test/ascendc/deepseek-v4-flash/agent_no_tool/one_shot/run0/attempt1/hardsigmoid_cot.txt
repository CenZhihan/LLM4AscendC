Input: x, float, ND. Output: y, float, ND. No attributes (no negative_slope). So project_json_src has no attr.

For tiling: UB budget similar, but compute per element: we need a tmp queue for intermediate? We can compute x/6 + 0.5 and then clamp. Use vector min/max or just compute with Mins, Maxs.
